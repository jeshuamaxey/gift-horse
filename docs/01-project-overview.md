## Project Description
This is a new Expo project built with React Native and Expo Router. The project is in its initial setup phase with a clean slate after moving boilerplate code to the `app-example` directory.

## Key Information
- The boilerplate/starter code has been moved to the `app-example` directory and can be used as a reference for implementing certain features
- The `app-example` directory contains template code that demonstrates:
  - Tab-based navigation setup
  - Theme/color scheme handling (light/dark mode)
  - Modal presentations
  - Custom hooks for color scheme
  - Themed components
  - UI components (collapsible, icon symbols)

## Development Environment
- **Platform**: Expo (~54.0.20)
- **Framework**: React Native (0.81.5)
- **Router**: Expo Router (~6.0.13)
- **Language**: TypeScript
- **New Architecture**: Enabled (`newArchEnabled: true`)

## Getting Started
1. Install dependencies: `npm install`
2. Start development server: `npx expo start`
3. Development is done in the `app` directory using file-based routing
